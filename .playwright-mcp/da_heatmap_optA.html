Reading Guide — Two views of the same data
Events Started (existing) — ชั่วโมงไหนมี DOWN event เปิดกี่ครั้ง
→ บอกว่า "เมื่อไหร่เครื่องเริ่มเสีย" แต่ไม่รู้ backlog
Machines in DOWN state (new) — ชั่วโมงไหนมีเครื่องค้าง down อยู่กี่เครื่อง
→ บอก "backlog" ของช่าง ณ ขณะนั้น — สูง = เครื่องรอซ่อมสะสม
